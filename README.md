# Bitácora Inversión - Beta 1.9

¡Bienvenido a la versión Beta 1.9! Esta actualización se centra en mejorar la experiencia del usuario nuevo y en profundizar en el análisis operativo de la cartera.

## 🚀 Novedades de esta Versión

### 1. Tour Interactivo de Bienvenida (Onboarding)
Hemos implementado una guía interactiva que se activa la primera vez que abres la aplicación.
- **Foco Dinámico**: La pantalla resalta los botones y menús clave mientras los explica.
- **Popovers Contextuales**: Tarjetas informativas que te guían paso a paso.
- **Claridad Visual**: Hemos mejorado el contraste y eliminado distracciones para que te centres en aprender la lógica de la app.

### 2. Reporte de Operaciones Cerradas
En el **Resumen Anual**, ahora encontrarás una nueva sección que detalla las **Operaciones Cerradas** del periodo.
- Identifica activos liquidados por completo.
- Calcula el beneficio/pérdida realizado real (Resultado Total).
- Muestra la rentabilidad final de cada inversión exitosa.

### 3. Optimización del Performance Hub
- **Zero-Scroll**: Hemos ajustado los márgenes y alturas de las gráficas para que todo el cuadro de mando quepa en una sola pantalla.
- **Separación de Gráficas**: Las gráficas de Evolución y Resultados ahora viven en tarjetas independientes dentro del panel, mejorando la legibilidad.

### 4. Mejoras Estéticas y de UI
- **Alineación del Menú Lateral**: Corregido el fallo de alineación del resaltado (hover/active) en la barra lateral.
- **Consistencia de Botones**: Alturas y espaciados normalizados para una sensación más fluida y profesional.

## 🛠️ Instrucciones de Uso
- Para ver el tutorial de nuevo, puedes resetear la configuración desde el apartado de **Configuración > Resetear Aplicación** (atención: esto borrará tus datos).
- Las **Operaciones Cerradas** aparecen automáticamente al generar el informe anual si has vendido el 100% de la posición de algún activo durante ese año.
